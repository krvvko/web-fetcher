How does it work?

Server side (Backend) 
app.get('/fetchWeb', async (req, res) => {
    const url = req.query.url;
    if (!url) {
        res.status(400).send('Missing URL parameter');
        return;
    }
    try {
        const response = await fetch(url);
        const html = await response.text();
        res.send(html);
    } catch (error) {
        console.log(error);
        res.status(500).send('Error fetching data');
    }
});

CORS and CORS policies

This is a security feature that allows the server to control which external sources are allowed to access its resources, so we cannot get the data of any site using the frontend only. Therefore, using the server part of the application, a request is sent to the required website, and information is fetched from it

Fetching

The frontend magic happens, and the data from Node.js is outputted into impromptu browser in browser
User side (Frontend) 
try {
    const response = await fetch(`/fetchWeb?url=${encodeURIComponent(url)}`);
    const html = await response.text();
    document.getElementById('content').innerHTML = html;
} catch (error) {
    console.log(error);
    alert('Error fetching data');
}
